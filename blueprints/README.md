# Example Platform Blueprints

This directory contains dummy blueprint content for a Kubernetes platform team operating with GitOps as the main delivery mechanism.

The examples describe cloud-specific environments and one cross-cutting production release-management flow.

Included examples:

- `gcp-gitops-event-platform.md`
- `aws-gitops-event-platform.md`
- `azure-gitops-event-platform.md`
- `production-release-management-flow.md`

These are intentionally generic, but written to resemble the type of architecture notes and operating blueprints a platform team would keep in Klaudia.
